## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user amount input → useUnwrap | Untrusted decimal string becomes a confidential-unit bigint; must be validated ≤ decrypted balance. |
| client → Zama KMS/relayer | The burn handle is publicly decrypted off-chain; the returned cleartext+proof are consumed by the finalize tx. |
| client → onchain wrapper | The unwrap + finalize txs cross to Sepolia; the wrapper enforces burn-from-caller and verifies the decryption proof. |

Persist pending on submit, clear only on finalize. |
