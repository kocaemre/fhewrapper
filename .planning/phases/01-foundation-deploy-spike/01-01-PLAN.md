## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| npm registry -> local build | Third-party package code executes at install/build (supply chain) |
| build config -> browser bundle | Values compiled into the client bundle are public by design |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
