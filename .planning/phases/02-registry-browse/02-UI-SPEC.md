## Spacing Scale

**Transcribed from the `.dc.html` engraving layout.** This design does NOT sit on a strict 8-point grid ‚Äî it uses hand-tuned values (14, 18, 30, 34) for the copperplate look. Deviating to "round" values breaks visual fidelity with the locked source, so these exact values are the contract.

| Token | Value | Usage (from source) |
|-------|-------|---------------------|
| 2xs | 6px | Icon-to-label gap in badges/copy buttons; filter-button group gap |
| xs | 7px | Address-row vertical gap inside card |
| sm | 8px | Baseline gap (symbol ‚áÑ symbol, address label‚Üíbutton) |
| ‚Äî | 10px | Filter row gap; card-header icon gap-ish |
| md | 12px | Copy-button padding-x; card-header gap; address-block padding-top |
| ‚Äî | 14px | **Card internal section gap** (`flex` gap on `<article>`) |
| lg | 18px | **Grid gap between cards**; card CTA padding-top |
| ‚Äî | 20px | **Card padding** (all sides) |
| xl | 24px | Toolbar (search+filter) bottom margin |
| ‚Äî | 30px | Header horizontal padding; main horizontal padding |
| ‚Äî | 34px | Hero banner bottom margin |
| 2xl | 40px | Main content top padding |
| 3xl | 120px | Main content bottom padding |

Exceptions: none beyond the above ‚Äî do not normalize these values.

## Layout & Responsive

- Page shell: sticky `Header` (Phase 1) ‚Üí optional wrong-network banner (Phase 1 `ChainGuard`) ‚Üí `<main>` `max-width:1180px; margin:0 auto; padding:40px 30px 120px`.

- Reduced motion: hover-lift transitions are decorative; keep the skeleton `inkPulse` subtle and respect `prefers-reduced-motion` (no essential animation this phase).

## Interaction Contract

| Interaction | Behavior |
|-------------|----------|
| Search typing | Live client-side filter across symbol/cSymbol/name/both addresses; empty result ‚Üí `RegistryEmpty` with the typed query echoed |
| Filter toggle | `all`/`valid`/`revoked`; active chip uses `--block` inverse fill; combines with search (AND) |
| Copy address | Click either address button ‚Üí clipboard write ‚Üí transient success toast; button hover tightens border to `--line` (ERC-7984 ‚Üí `--blue`) |
| Card hover (valid) | `transform:translate(-2px,-2px)` + hard `--shadow` |
| Wrap CTA | Valid ‚Üí navigates to the (future) wrap panel for that pair; **this phase may route to a placeholder/disabled target** since wrap ships in Phase 4 ‚Äî CTA remains visible per the locked design. Revoked ‚Üí disabled, `Unavailable` |
| Data load | Mount ‚Üí skeleton grid ‚Üí onchain multicall resolves ‚Üí render cards; multicall/RPC error ‚Üí `RegistryError` retry |
| Theme toggle | Existing Phase-1 mechanism flips `data-theme` between `parchment`/`cellar`; all tokens follow via CSS vars |
