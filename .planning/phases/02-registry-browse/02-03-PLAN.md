## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user input -> client filter | Search string used only for client-side string matching (no query/eval) |
| public RPC -> client | Error/loading states derived from react-query read status |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-06 | Injection | search string used in filtering | low | mitigate | Search is a plain case-insensitive substring match over in-memory fields; never interpolated into HTML, a query, or eval; echoed into empty-state via React escaping |
| T-02-03 | Tampering | RPC read failure surfaces raw error | low | mitigate | RegistryError shows a fixed human-readable message + Retry(refetch); raw revert/error strings are not rendered |
| T-02-01 | Tampering/Injection | empty-state echo of the typed query | low | mitigate | React default escaping of the echoed search text |
</threat_model>
